## 🛠 Troubleshooting

- If the vault does not open correctly, make sure you selected the extracted folder and not the ZIP file.
- For habit tracking tables to work, enable Obsidian's core plugin called “Tasks” or the one recommended in the vault notes.
- If charts do not show, check if Obsidian supports embedded content on your system.
